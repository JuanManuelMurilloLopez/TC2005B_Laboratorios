¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet es la red compuesta de computadoras interconectadas, mientras que la World Wide Web es el conglomerado de las páginas disponibles para las computadoras de la red [1]

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  1. GET
    Utilizado para acceder a datos del servidor
  2. HEAD
    Utilizado para recuperar los encabezados
  3. POST
    Se utiliza para crear nuevos registros de datos
  4. PUT
    Se utiliza para actualizar información ya existente en el servidor
  5. PATCH
    Al igual que PUT se utiliza para actualizar información, pero con la funcionalidad para evitar sobreescritura de datos indeseada
  6. DELETE
    Utilizado para eliminar registros de la base de datos

[3]

Un servidor web devuelve una respuesta con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

El código de respuesta 200 es OK, lo que significa que todo está corriendo de manera deseada y no hay ningún error

¿Es la responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró un error 404? ¿Por qué?

El código 404 significa No Encontrado, el desarrollador deberá verificar que el sitio esté corriendo de manera correcta en el servidor.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró un error 500? ¿Por qué?

El código HTTP 500 refiere a un "Internal Server Error", esto ocurre cuando el servidor no puede procesar la solicitud, por lo que el desarrollador deberá verificar el estado del servidor

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado? Menciona algunos elementos de HTML4 que en HTML5 estén desaprobados

Un atributo HTML que está depreciado quiere decir que ya no se recomienda su uso ya que posiblemente los navegadores ya no lo soporten proximamente
Ejemplos de etiquetas depreciadas:
  • dir
  • hgroup
  • menu
  • listing

    ¿Cuáles son las diferencias principales entre HTML4 y HTML5?

  • HTML5 tiene mayor compatibilidad con los navegadores a comparación de su predecesor
  • HTML5 cuenta con soporte multimedia incluyendo audio, video, animaciones, etc.
  • Eliminación del estándar SGML
  • HTML5 tiene un mejor rendimiento
